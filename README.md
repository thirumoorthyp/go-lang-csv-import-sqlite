# go-lang-import-csv-into-sqlite
This go-lang code is used to import data from CSV file into the SQLite database.

# Manual Steps: 
``` shell
apt install golang
```

``` shell
go version
```
# go version go1.20.3

# config
``` shell
 set GOOS=linux
 set GOARCH=amd64
 set CGO_ENABLED=0
```

# install sqlite3
``` shell
go install sqlite3
```

# executing go program
```shell
go run main.go
```
